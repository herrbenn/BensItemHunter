# Bens Item Hunt
ItemHunter ist eine Challenge für deinen Minecraft-Server: Starte eine gemeinsame Jagd, bei der Spieler nach und nach alle möglichen Items sammeln, alle Mobs besiegen und alle Advancements/Achievements abschließen müssen – inklusive Timer und Live-Fortschritt.

![Item Hunt Banner](https://cdn.modrinth.com/data/cached_images/e4fd5594ff8309bd0be5100bd3692cb2cfe15b8d.png)

## Features (kurz):

**Kombinierte Challenge**: 
Items finden + Mobs töten + Advancements abschließen (Gesamtfortschritt zählt zusammen)

**Ingame-UI**: 
Übersicht per GUI (Items/Challenge-Ansicht) und Status-Anzeige
Timer-System mit Pause/Resume (pausiert automatisch, wenn niemand online ist)

**Live-Anzeigen**: 
ActionBar-Timer, BossBar-Fortschritt und optional Tablist mit TPS/Ping + Fortschritt

**Konfigurierbar**: 
Item-Kategorien aktivieren/deaktivieren, zusätzliche Items ausschließen, Sounds & Prefix anpassen

**Admin-Command**: 
/itemhunt mit Start/Stop/Pause/Resume/Status/Items/Reset/Skip
Folia-Support (läuft auch auf Paper/Bukkit)
